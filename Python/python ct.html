Q1 (a) Describe Membership and Identity operators in python with appropriate example. [7 Marks]

1. Identity Operators

Identity operators compare the memory locations of two objects to check if they are exactly the same object, not just if they have the same value.

OperatorDescriptionExample
is Returns True if both variables point to the same object. x is y
is not Returns True if both variables point to different objects. x is not y

2. Membership Operators

Membership operators are used to test if a sequence (string, list, tuple, set, dictionary) contains a specific value or item.

OperatorDescriptionExample
in Returns True if a specified value is present in the object. x in y
not in Returns True if a specified value is NOT present in the object. x not in y 
# --- Identity Example ---
a = [1, 2, 3]
b = [1, 2, 3]
c = a

print(a is c)      # Same object in memory
print(a is b)      # Different objects, even though values match
print(a == b)      # Values match

# --- Membership Example ---
fruits = ["apple", "banana"]
print("apple" in fruits)
print("mango" not in fruits)

False

True

Q2 (a) Explain different loops available in python with suitable examples. [7 Marks]

Python provides two primary looping constructs to execute a block of statements repeatedly.

1. The for Loop

Used for iterating over a sequence (such as a list, tuple, dictionary, set, string) or a range of numbers. It automatically handles the iteration and stops when the sequence is exhausted.

# Example: for loop using range
for i in range(1, 4):
    print("Iteration:", i)
Terminal Output
Iteration: 1
Iteration: 2
Iteration: 3

2. The while Loop

Executes a set of statements continuously as long as a specified boolean condition is True. The condition must eventually become False, or it will result in an infinite loop.

# Example: while loop
count = 1
while count <= 3:
    print("Count is:", count)
    count += 1  # Increment counter to avoid infinite loop
Terminal Output
Count is: 1
Count is: 2
Count is: 3

Q1 (a) What are Lists? Explain append(), extend(), pop() methods of list with suitable examples of each. [7 Marks]

Lists are ordered, mutable (changeable) collections of items enclosed in square brackets []. They can hold items of heterogeneous data types.

Method Description Example
append(item) Adds a single element to the very end of the list. It increases the length of the list by exactly one. L = [1, 2]
L.append(3) # L becomes [1, 2, 3]
extend(iterable) Adds all elements of an iterable (like another list, tuple, or string) to the end of the current list. L = [1, 2]
L.extend([3, 4]) # L becomes [1, 2, 3, 4]
pop(index) Removes and returns the element at the specified index. If no index is provided, it removes and returns the last item. L = [10, 20, 30]
L.pop(1) # Returns 20, L becomes [10, 30]
Q1 (b) Write a python program to count uppercase and lowercase In given string "Python is FUN " [3 Marks] 
text = "Python is FUN "
upper_count = 0
lower_count = 0

for char in text:
    if char.isupper():
        upper_count += 1
    elif char.islower():
        lower_count += 1

print("Uppercase count:", upper_count)
print("Lowercase count:", lower_count)
Terminal Output
Uppercase count: 5
Lowercase count: 7

Q2 (a) Define Tuples. Demonstrate How to change tuple values? [7 Marks]

Definition

A Tuple is a sequence data type in Python, similar to a list, but it is immutable. Once a tuple is created, its elements cannot be added, removed, or modified. They are enclosed in parentheses ().

How to change tuple values

Because tuples are immutable, you cannot directly assign a new value to an index (e.g., t[0] = 5 will raise an error). To alter a tuple, you must perform a workaround:

  1. Convert the tuple into a list.
  2. Modify the desired values in the list.
  3. Convert the list back into a tuple.
# Original Tuple
my_tuple = ("Apple", "Banana", "Cherry")
print("Original:", my_tuple)

# 1. Convert to List
temp_list = list(my_tuple)

# 2. Modify the List
temp_list[1] = "Blueberry"  # Changing 'Banana' to 'Blueberry'
temp_list.append("Orange")  # Adding a new item

# 3. Convert back to Tuple
my_tuple = tuple(temp_list)

print("Modified:", my_tuple)
Terminal Output
Original: ('Apple', 'Banana', 'Cherry')
Modified: ('Apple', 'Blueberry', 'Cherry', 'Orange')
