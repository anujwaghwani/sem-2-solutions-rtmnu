Question 1

1 (A) Explain the working of input() function and eval() function in brief alongwith an example. [7 Marks]

1. The input() Function

The input() function is used to take user input from the keyboard. It halts the program until the user enters data. Crucially, it always reads and returns the input as a string (str), so type casting is needed for math.

2. The eval() Function

The eval() function parses a string and executes it as a Python expression. When used with user input, it automatically detects and evaluates the string into its corresponding Python datatype (like an int, float, or list).

# input() example
age = input("Enter age: ") # User types 20
print(type(age))           # Output is string

# eval() example
data = eval("5 * 4")
print("Evaluated data:", data)
print(type(data))          # Output is int
Terminal Output
<class 'str'>
Evaluated data: 20
<class 'int'>

A function is a reusable block of code that performs a specific task. It is defined using the def keyword, followed by the function name, parentheses (which may contain parameters), and a colon. The code block is indented.

print("The sum is:", result)

6 (A) Describe the role of return statement in a Python function. Illustrate the working of return statement alongwith an example. [7 Marks]

The return statement serves two primary roles in a function:

  1. It immediately terminates the execution of the function.
  2. It passes a value (or multiple values as a tuple) back to the caller. If no expression is given, it returns None.
def square_number(num):
    result = num * num
    return result # Passes value back and exits
    print("This will never print")

output = square_number(6)
print("Returned value:", output)
Terminal Output
Returned value: 36

Question 8 (OR)

8 (A) Explain the working of zip() functions in Python alongwith an example. [7 Marks]

The zip() function takes two or more iterables (like lists, tuples, or strings) and pairs their elements together based on their index. It returns an iterator of tuples, where the i-th tuple contains the i-th element from each of the argument iterables. The pairing stops when the shortest iterable is exhausted.

names = ["Alice", "Bob", "Charlie"]
ages = [24, 30, 22]

# Zipping two lists together
paired_data = zip(names, ages)

# Converting to a list to view the pairs
result_list = list(paired_data)
print("Zipped Result:", result_list)

# Common usage in a loop
for name, age in zip(names, ages):
    print(f"{name} is {age} years old.")
Terminal Output
Zipped Result: [('Alice', 24), ('Bob', 30), ('Charlie', 22)]
Alice is 24 years old.
Bob is 30 years old.
Charlie is 22 years old.
8 (B) Describe "in" and "not in" operators in sets. Illustrate with an example. [7 Marks]

These are Membership Operators used to check if a specific element exists within a collection. In Python Sets, these operators are highly optimized (using hash tables) and operate in O(1) constant time, making them much faster than using "in" on a list.

  • in: Returns True if the element exists in the set.
  • not in: Returns True if the element does NOT exist in the set.
my_set = {"apple", "banana", "cherry"}

# Using 'in'
print("Is 'apple' in set?", "apple" in my_set)

# Using 'not in'
print("Is 'mango' not in set?", "mango" not in my_set)
print("Is 'banana' not in set?", "banana" not in my_set)
Terminal Output
Is 'apple' in set? True
Is 'mango' not in set? True
Is 'banana' not in set? False

Question 9

9 (A) Construct a program in Python to write a text into a file. [7 Marks]

To write text, we use the open() function in write mode ('w'). If the file doesn't exist, it is created. If it does exist, it gets overwritten. We use the with statement to ensure the file closes automatically.

# Program to write text into a file
file_name = "sample.txt"

# Open file in write mode
with open(file_name, 'w') as file:
    file.write("Hello, this is the first line.\n")
    file.write("Python File Handling is easy!")

print(f"Data successfully written to {file_name}")
Terminal Output
Data successfully written to sample.txt
9 (B) Explain how numeric data can be written to a file in Python. Demonstrate with a program. [7 Marks]

The write() method strictly requires a string argument. Therefore, numeric data (like integers or floats) cannot be written directly. They must first be converted into strings using the str() function or string formatting before being passed to write().

scores = [85, 90, 78, 92]

with open("scores.txt", "w") as file:
    for score in scores:
        # Convert numeric data to string before writing
        file.write(str(score) + "\n")

print("Numeric scores written as text.")
Terminal Output
Numeric scores written as text.

10 (A) Design a program in Python to read integers line by line and calculate their sum. [7 Marks] 
# Assuming 'numbers.txt' exists and contains numbers line by line
# E.g., 10 \n 20 \n 30

total_sum = 0

try:
    with open("numbers.txt", "r") as file:
        for line in file:
            # strip() removes \n, int() converts the string to an integer
            num = int(line.strip()) 
            total_sum += num
            
    print("The total sum of numbers is:", total_sum)
except FileNotFoundError:
    print("File not found! Please create 'numbers.txt'.")
10 (B) Write a program in Python where new text is added without overwriting existing content. [7 Marks]

To add content without overwriting, the file must be opened in Append mode ('a'). In this mode, the file pointer is placed at the very end of the file, and any new data written is added there.

# Opening the file in append mode ('a')
with open("sample.txt", "a") as file:
    # Adding a new line of text at the end of the file
    file.write("\nThis line is appended without overwriting.")

print("New text successfully appended!")
Terminal Output
New text successfully appended!
