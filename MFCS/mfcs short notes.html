1. Sets & Relations

Fundamental Properties of Relations

A relation $R$ on a set $A$ is a subset of $A \times A$. The core properties defining relation classifications are:

  • Reflexive: Every element relates to itself. $(a, a) \in R$ for all $a \in A$.
    Matrix: Main diagonal consists entirely of 1s. Digraph: Every node has a self-loop.
  • Symmetric: Direction doesn't matter. If $(a, b) \in R \implies (b, a) \in R$.
    Matrix: $M_R = (M_R)^T$. Digraph: Every edge has a returning edge (bidirectional).
  • Transitive: Chain reactions. If $(a, b) \in R$ and $(b, c) \in R \implies (a, c) \in R$.
  • Anti-Symmetric: One-way streets. If $(a, b) \in R$ and $(b, a) \in R \implies a = b$.

Important Composites

Equivalence Relation: Reflexive + Symmetric + Transitive. (Partitions a set into disjoint equivalence classes, e.g., congruence modulo n).

Partial Order Relation (POSET): Reflexive + Anti-Symmetric + Transitive. (Creates a hierarchy, e.g., subset relation $A \subseteq B$ or "divides" relation).

Fig 1: Digraph of a relation. Blue edges show original $R = \{(1,1), (2,2), (1,2), (2,3)\}$. The red dashed edge $(1,3)$ must be added to create the Transitive Closure ($R^*$). Loops show reflexivity.

Operations on Relation Matrices

Let $M_R$ and $M_S$ be boolean relation matrices (containing only 0s and 1s). Operations are performed using boolean arithmetic ($1 \wedge 1 = 1$, $1 \vee 1 = 1$, $1+1=1$).

  • Inverse ($R^{-1}$): Just transpose the matrix. $M_{R^{-1}} = (M_R)^T$.
  • Union ($R \cup S$): Element-wise logical OR ($\vee$). $M_{R \cup S} = M_R \vee M_S$.
  • Intersection ($R \cap S$): Element-wise logical AND ($\wedge$). $M_{R \cap S} = M_R \wedge M_S$.
  • Composition ($R \circ S$): Boolean matrix multiplication. To find $(i,j)$th element, take the boolean dot product of the $i$th row of $M_R$ and $j$th column of $M_S$.

2. Functions, Characteristics & Fuzzy Sets

Types of Mappings

A function maps elements from a Domain ($X$) to a Codomain ($Y$). The actual mapped values form the Range.

Injective (One-to-One)

Every $x$ maps to a unique $y$. Not all $y$ need to be mapped.

Surjective (Onto)

Every $y$ is mapped to by at least one $x$. Range = Codomain.

Bijective (Both)

Perfect 1-to-1 matching. Only Bijective functions have an Inverse ($f^{-1}$).

Characteristic Functions

A mathematical way to define subsets. $f_A(x) = 1$ if $x$ is in the set $A$, and $0$ if it is not. This converts set operations into binary arithmetic:

  • Intersection: $f_{A \cap B} = f_A \cdot f_B$ // Only 1 if BOTH are 1 ($1 \cdot 1 = 1$)
  • Union: $f_{A \cup B} = f_A + f_B - (f_A \cdot f_B)$ // Subtract the intersection to avoid double counting
  • Complement: $f_{A'} = 1 - f_A$ // Flips 1 to 0, and 0 to 1

Fuzzy Sets

Traditional sets are black and white (element is in or out). Fuzzy sets allow "degrees of membership" denoted by $\mu_A(x) \in [0, 1]$. Operations mirror standard sets but use Min/Max functions:

  • Union ($A \cup B$): Take the Maximum. If $x$ is $0.8$ in $A$ and $0.5$ in $B$, it is $\max(0.8, 0.5) = 0.8$ in the union.
  • Intersection ($A \cap B$): Take the Minimum. The overlap is limited by the smallest membership. $\min(0.8, 0.5) = 0.5$.
  • Complement ($A'$): Standard inversion: $1 - \mu_A(x)$.

3. Curve Fitting & Statistical Methods

Method of Least Squares (Curve Fitting)

The goal is to find a curve that minimizes the sum of the squares of the errors (the vertical distances between actual points and the curve). For non-linear curves, we use logarithms to transform them into linear forms ($Y = A + BX$).

Curve Type Original Equation Transformation Linear Form ($Y = A + BX$)
Parabola $y = a + bx^2$ Let $X = x^2, Y = y$ $Y = a + bX$
Exponential $y = ae^{bx}$ Take $\ln$: $\ln y = \ln a + bx$ $Y = A + bX \quad (\text{where } A = \ln a, Y = \ln y)$
Geometric $y = ax^b$ Take $\log_{10}$: $\log y = \log a + b \log x$ $Y = A + bX \quad (\text{where } A = \log a, X = \log x)$

The Normal Equations (for $Y = A + BX$):

$\sum Y = nA + B \sum X$
$\sum XY = A \sum X + B \sum X^2$

Correlation & Regression Analysis

  • Karl Pearson’s Correlation Coefficient ($r$): Evaluates the linear relationship between two variables. Ranges from $-1$ (perfect negative correlation) to $+1$ (perfect positive).
    $$r = \frac{\sum (x-\bar{x})(y-\bar{y})}{\sqrt{\sum (x-\bar{x})^2 \cdot \sum (y-\bar{y})^2}} = \frac{SS_{xy}}{\sqrt{SS_{xx} \cdot SS_{yy}}}$$
  • Regression Lines: The two lines of best fit.
    1. $y$ on $x$ (Predict $y$ given $x$): $y - \bar{y} = b_{yx}(x - \bar{x})$. Slope $b_{yx} = \frac{SS_{xy}}{SS_{xx}}$.
    2. $x$ on $y$ (Predict $x$ given $y$): $x - \bar{x} = b_{xy}(y - \bar{y})$. Slope $b_{xy} = \frac{SS_{xy}}{SS_{yy}}$.
  • Spearman's Rank Correlation ($\rho$): Used when data is ranked or non-parametric.
    $$\rho = 1 - \frac{6(\sum d^2 + CF)}{n(n^2 - 1)}$$
    Crucial: If multiple items tie for a rank, assign them the average rank and add a Correction Factor (CF) to the formula: $CF = \sum \frac{m(m^2-1)}{12}$ where $m$ is the number of tied items for each rank block.

4. Abstract Algebra

Hierarchy of Algebraic Structures

Algebra studies sets equipped with operations (like $+$, $\times$, $\circ$). The structures are defined by how many axioms they satisfy. Each structure builds upon the previous one.

Fig 3: The nested hierarchy of algebraic systems. A Group must satisfy all 4 core axioms.

Key Definitions & Axioms

  • Identity Element ($e$): An element that leaves other elements unchanged. $a * e = a$. (e.g., $0$ for addition, $1$ for multiplication, Identity Matrix $I$ for matrix multiplication). Required for a Monoid.
  • Inverse Element ($a^{-1}$): The element that "cancels out" $a$ to produce the identity. $a * a^{-1} = e$. Required for a Group.
  • Two-Step Subgroup Test: To prove $H$ is a subgroup of $G$, show: (1) $H$ is closed under the operation ($a,b \in H \implies ab \in H$) and (2) Every element in $H$ has its inverse in $H$ ($a \in H \implies a^{-1} \in H$).
  • Normal Subgroup ($H \triangleleft G$): A highly symmetric subgroup where left and right cosets coincide. Proof condition: For any $g \in G$ and $h \in H$, the conjugate $ghg^{-1}$ must be inside $H$. Note: The intersection of any two normal subgroups is always a normal subgroup.

Homomorphism

A structure-preserving mapping between two groups. A function $f: G \to G'$ is a homomorphism if $f(a * b) = f(a) \star f(b)$. Two critical properties you must be able to prove:

  1. $f(e) = e'$: The identity maps to the identity.
  2. $f(a^{-1}) = [f(a)]^{-1}$: The inverse of the input maps to the inverse of the output.

5. Combinatorics & Recurrence Relations

Combinatorics Rules

  • Rule of Product (AND): If task A can be done in $m$ ways AND task B in $n$ ways, doing both takes $m \times n$ ways.
  • Rule of Sum (OR): If task A can be done in $m$ ways OR task B in $n$ ways (mutually exclusive), doing either takes $m + n$ ways.
  • Combinations ($^nC_r$): Selecting groups where order doesn't matter (e.g., picking marbles). $^nC_r = \frac{n!}{r!(n-r)!}$.
    Exam Strategy: For "at least X" problems, split the problem into mutually exclusive cases (e.g., exactly 2, exactly 3, exactly 4), calculate combinations for each case using the product rule, and sum the cases together.

Generating Functions

A powerful tool that converts a sequence of numbers into a single polynomial function. $G(x) = a_0 + a_1x + a_2x^2 + \dots$. They are solved using the infinite geometric series identity:

$$\frac{1}{1-rx} = 1 + rx + (rx)^2 + (rx)^3 + \dots$$

Sequence: $1, 1, 1, 1, \dots$

Substitute $r=1 \implies G(x) = \frac{1}{1-x}$

Sequence: $1, -1, 1, -1, \dots$

Substitute $r=-1 \implies G(x) = \frac{1}{1+x}$

Sequence: $1, 0, -1, 0, 1 \dots$

This is $1 - x^2 + x^4 - x^6 \dots$ Substitute $rx = -x^2 \implies G(x) = \frac{1}{1+x^2}$

Sequence: $0, 0, 1, 1, 1 \dots$

Factor out $x^2(1+x+x^2\dots) \implies G(x) = \frac{x^2}{1-x}$
